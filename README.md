# Juanma

**Security Engineer** — 8+ years in cybersecurity, from application and product security to payments and cryptographic infrastructure.

I've spent my career securing high-growth platforms across LATAM: standing up vulnerability management from scratch, running bug bounty and pentest programs, hardening payment systems and HSM/key infrastructure, and more recently focusing on the security of AI-generated code and LLM-powered products. I like understanding systems well enough to break them, and automating the security work that shouldn't be manual.

## Focus areas

- **Product & application security** — vulnerability management, threat modeling, secure SDLC
- **AI / LLM security** — prompt injection, abuse scenarios, guardrails for AI-generated code
- **Cryptography & payments** — HSM & key management, AWS Payment Cryptography, PCI DSS
- **Cloud security** — AWS, CNAPP/CSPM, IaC, access governance
- **Compliance** — PCI DSS v4.0, SOC 2 Type II, ISO 27001, LATAM financial regulations

## Background

Led security for **Mercado Pago** (Mercado Libre) payment infrastructure across five LATAM countries, and built internal security tooling — a Go/Python DLP platform, compliance automation — years before that kind of tooling became standard. Earlier, IT solution architecture at Telefónica.
